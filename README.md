# clockin

 一个基于vue3的简单打卡项目

# 运行环境
    - node version > 16.10.0

## 项目设置

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
# 功能支持
 - [ ] 本地脚本打卡 🚧
 - [ ] 可配置数据库 🚧
